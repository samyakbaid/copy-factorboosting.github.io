Cross-Sectional Portfolio Analysis

Fama-French factor portfolios on Indian equities — compare up to 4 strategies

How to Use

This tool lets you build and compare portfolios based on Fama-French factors. Select characteristics like size, value, momentum, profitability, and investment style to see how a portfolio of stocks matching those traits would have performed historically.

Quick start guide
  1. Pick factor labels — In the sidebar, click one or more buttons (e.g., "Small" under Size, "Winner" under Momentum). These define which stocks go into your portfolio.
  2. Click "Run Analysis" — This builds the portfolio and shows the performance chart, statistics, and stock holdings.
  3. Compare strategies — To compare multiple portfolios, click "+ Add Portfolio" to save your current selection, pick new portfolios, add again, then click "Run Comparison". You can compare up to 4 portfolios side by side.
  4. Explore holdings — Use the slider below the chart to scrub through months and see exactly which stocks were in the portfolio at any point in time. Hovering on the chart also updates the holdings panel.
  5. Switch weighting — Toggle between Equal Weighted (every stock gets the same weight) and Value Weighted (larger companies get more weight) using the buttons above the chart.

Tip: For a Long/Short strategy, switch to "Long / Short" mode, pick your long-side factors, then add short-side factors separately. The return shown is the long portfolio return minus the short portfolio return.


Note: The Transaction cost model is more accurate for Equal Weighted (EW) portfolios as it only tracks the stocks changed, not how much weight shifted. We're working on implementing a more realistic model for Value Weighted (VW) portfolios as well.
